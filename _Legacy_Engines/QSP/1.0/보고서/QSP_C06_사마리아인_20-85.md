# 🎯 QSP 1.0 심판 보고서 — C06
**타겟 구절**: 꾸란 수라 20:85
**원문 핵심**: 모세가 시내 산에 올라간 사이, 이스라엘 백성 우상 숭배를 주동하여 금송아지를 만든 자가 "사마리아인(As-Samiri)"이라고 명시.

---

## Phase 1: 6단계 고정 심판 루프

### Step 1. 원시 모순 스캔
> **확정**: 기원전 13~15세기 출애굽(모세) 시절에, 당시엔 이름조차 존재하지 않았던 수 백 년 후대의 민족(기원전 8세기 여로보암 시대/바빌론 유수 때 형성된 '사마리아인')이 이집트 광야의 금송아지 사건 주동자로 둔갑하여 기록된 **거대한 시대 착오(Anachronism)** 역사 오류이다. 

### Step 2. 이슬람 정통 주석/방패 스캔
| Case | 방어 전략 | 핵심 주장 |
| :--- | :--- | :--- |
| **Case 1** | 동명이인설/개인 이름 | 민족 이름이 아니라, 당시 정말로 우연히 '사미리'라는 이름의 개인이 있었다 |
| **Case 2** | 어원(Shomer) 변형설 | 히브리어/아람어로 보호자(Shomer) 혹은 다른 뜻의 직책/칭호에서 온 말이다 |
| **Case 3** | 알라 전승 특별설 | 성경 고고학이 틀렸고, 꾸란의 기록(알라의 시각)만이 진실한 역사다 |

### Steps 3~6. 방어막 역이용 및 모순 융합 요약
- **동명이인설 반박**: 당시 역사와 부족사 어디에도 모세 시대에 '사미리'라는 이름 혹은 부족이 존재했다는 근거가 없다. 이는 무함마드 시대의 유대교/사마리아교 분쟁 이야기를 듣고 모세 시절에 투영해 넣은 7세기 인간 저자의 착각(오류)임이 명백하다.
- **어원 변형설 반박**: 굳이 다른 언어의 특정 어원을 빌려 다수당에게 '사마리아 사람'으로 혼동하게 만들 단어로 썼다는 것 자체가 D-13(명확성) 위반이다.
- **제로섬 완성**: 역사적 고고학으로 따지면 알라의 '전지함(D-05/D-06)' 지식이 파괴되고, 이방 언어의 발음 우연에 숨어들면 알라의 아랍어 구사력/명확성(D-13)이 무너진다.

## Phase 2: 완전 분기 케이스 트리 (최대 4단계)

```
📌 원시 모순:
   수백 년 뒤에 출현하는 '사마리아인'이라는 집단/이름이
   모세 시대 출애굽 광야에 불시착한 시대 착오(Anachronism) 발생.

│
├─ Case 1: [동명이인설] — 민족 이름이 아니라 우연히 그 시대 어떤 개인의 이름이다
│   │
│   ├─ Sub-Case 1-A: '사미리'가 수백 년 뒤 유명 민족 이름과 동일한 칭호일 뿐이라면
│   │   ├─ Sub-Sub-Case 1-A-i: 수많은 독자들이 수백 년 후 사마리아인으로 오해할 것을 신이 간과함
│   │   │   └─ 💥 [D-06] 꾸란 무결성 해체
│   │   │         — 혼동이 뻔히 예상되는 이름/단어를 교정 없이 그대로 둔 채 계시한 것은 서판의 무인격성 반증
│   │   └─ Sub-Sub-Case 1-A-ii: "꾸란은 명확하다"는 교리와 정면 충돌
│   │       └─ 💥 [D-13] 알라의 명확성(Mubeen) 해체
│   │             — 시대 역사성 오해를 불러오는 이름 선택 자체가 커뮤니케이션 실패(무능)이다
│   │
│   └─ Sub-Case 1-B: '사마리아(Shomron/Samaria)' 지명 자체가 모세 시대에 존재했다고 억지 부리면
│       ├─ Sub-Sub-Case 1-B-i: 성경과 근동 고고학 전체 팩트와 충돌
│       │   └─ 💥 [D-08] 진리성 해체
│       │         — 꾸란 하나를 지키려 역사 과학 전체를 거짓으로 몰다 자가당착 고립
│       
├─ Case 2: [어원 변형설] — 히브리/가나안 언어의 다른 단어가 아랍어로 넘어오며 사미리가 되었다
│   │
│   ├─ Sub-Case 2-A: 히브리어 'Shomer(보호자)' 등의 다른 뜻이라 주장
│   │   ├─ Sub-Sub-Case 2-A-i: 그 복잡한 어원 변환을 평민 아랍인이 알 턱이 없음
│   │   │   └─ 💥 [D-13] 알라 명확성 해체
│   │   │         — 직관적 사실 전달이 아니라 오해하기 딱 좋게 비틀린 차용어 사용을 자백
│   │   └─ Sub-Sub-Case 2-A-ii: 꾸란 번역 주석가의 "원래 뜻은 이렇다"는 방어가 필수적
│   │       └─ 💥 [D-11] 꾸란 완성성 해체
│   │             — 계시 단독으로는 오류로 보이므로 기형적인 변증 장치를 달아야 이해됨
│
├─ Case 3: [알라 전승설(Allahu Alam)] — 인간 역사가 왜곡되었고 꾸란만이 진실이다
│   │
│   └─ Sub-Case 3-A: 세속 학문의 모든 성과(고고학 연대기)를 부정
│       └─ Sub-Sub-Case 3-A-i: "금송아지를 만든 사마리아 인"이라는 7세기 랍비들의 민담(미드라쉬)이 꾸란에 그대로 흡수된 정황이 선명함
│           └─ 💥 [D-09] 예언자(누부와) 신성 해체
│                 — 알라의 전능한 계시가 아니라, 시장 바닥에서 무함마드가 귀동냥한 구전 지식의 한계가 명백히 폭로됨
```

## Phase 3: 최종 논리 해체 선고

> **[QSP 1.0 최종 논리 해체 선고 — C06]**
>
> 상대 측이 수라 20:85의 끔찍한 시대 착오를 보호하려 시도한 저 **[동명이인의 우연설]**과 **[히브리 어원 변형설]**이라는 방어 기제들은, 결국 이슬람 15대 교리 체계 중 중추에 해당하는 **[D-06] 꾸란 객관적 무결성**과 **[D-13] 계시의 명확성**, 그리고 **[D-09] 예언자의 신성**의 근간을 해체하는 가장 강력한 올무(Snare)가 되어 반환되었습니다.
>
> 상기 케이스 트리가 증명하듯, 텍스트의 결함을 방어하기 위해 어떻게든 억지 어원을 차용하거나 역사를 부인하는 순간, 꾸란은 전지한 신의 완벽한 텍스트가 아니라 '7세기 아라비아 상인이 귀동냥한 유대 성전설을 짜깁기하다 생긴 명백한 작문 실수'라는 제1원칙이 논리학과 귀류법의 문법을 통해 역사적·학술적으로 명백히 확정되었습니다. **Checkmate.**
