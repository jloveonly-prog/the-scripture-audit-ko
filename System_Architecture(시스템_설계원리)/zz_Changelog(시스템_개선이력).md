## 🆕 v2.1 변경 이력 (2026-05-16)

### 변경 1: TYPE-W 심화 확장 — 회고적 기술 논증 장착

**배경**: 베드로 갈보리 순교설 검증 중 "명시 vs 추론" 논쟁에서 도출된 새 논증.

**핵심 발견**:
> 요한은 AD 85-95년에 복음서를 기록했다. 요 13:36의 ὅπου를 선택할 때, 골고다는 이미 수십 년 전 완료된 사건이었다. 요한은 **알면서** ὅπου를 썼다. 따라서 ὅπου → τόπος(골고다) 연결은 독자의 사후 추론이 아닌 **저자의 의도적 설계**다.

**수정 파일**:
- 04_QUIVER(무기고)/TYPE-W_PropheticForeshortening.md — 심화 섹션 추가
- 03_WAR_LOG(전투)/[F+E+G+N+P+I+T+S]_베드로_갈보리순교설.md — TYPE-W 심화 발동 섹션 추가
- BVCAP_사용자_가이드.md — 3도메인 구조 섹션 추가
- BVCAP_User_Guide.md — 3-Domain Hybrid System 섹션 추가

### 변경 2: 3도메인 구조 공식화

**배경**: BVCAP 무기들이 해석학 · 논리학 · 오류학 세 도메인으로 구성됨을 발견. 사용자 가이드에 공식 정리.

**내용**:
- 해석학 = 본문에서 증거를 추출하는 도구 (F, G, W, S, I, T, R)
- 논리학 = 증거로 결론을 확정하는 도구 (N, P, L, AC, E)
- 오류학 = 상대 논증의 결함을 찾아 무력화하는 도구 (T, R, P 겸용)
- COMBO = 해석학+논리학 동시 발화 → IRONCLAD 생성 원리

## 🔬 v2.6 — 시스템 한계 분석 및 범용화 개선 로드맵 (2026-05-17)

### 📌 핵심 발견: BVCAP의 구조적 한계 2개

> 이 분석은 "BVCAP을 성경을 믿지 않는 일반인도 사용할 수 있는가?"라는 질문에서 출발했다.

#### 한계 1 — OVERRIDE-0 공리 문제

**현상**: BVCAP의 OVERRIDE-0은 "KJV 성경의 무오성"을 고정 공리로 설정한다.
**문제**: 이 공리를 수용하지 않는 상대(비기독교인, 다른 번역본 사용자, 학자)는
논증 시작점 자체를 거부할 수 있다.

```text
현재 구조:
  [공리: KJV 무오성] ──→ [논증 체계 전체] ──→ [판결]
        ↑
  이 공리를 거부하면 전체 논증이 외부에서 시작됨
```

#### 한계 2 — IRONCLAD 판결 과잉 주장

**현상**: IRONCLAD = "논리적 필연성"으로 선언.
**문제**: 논리적 필연성은 공리 체계 내부에서만 성립한다.
공리를 공유하지 않는 상대에게는 "당신의 전제 안에서만 필연"으로 들린다.

---

### ✅ 핵심 이해 (검증 완료) 

> **사용자 질문**: "시작 전 공리를 정하고 결과를 내는 구조라면, 다양한 사람들도 사용 가능하다는 것인가?"

**답: 정확히 맞습니다.**

BVCAP의 파이프라인 로직 자체(해석학 → 논리학 → 오류학)는 **도메인 독립적**입니다.
"어떤 공리를 전제로 시작하는가"만 바꾸면, 동일한 파이프라인으로
성경 이외의 모든 텍스트·논제를 분석할 수 있습니다.

```text
현재 (고정 공리):
  공리 = KJV 무오성 → 파이프라인 → 판결

개선 후 (선택형 공리):
  공리 L0 (보편) = "텍스트는 내부적으로 일관될 수 있다"      → 누구나 사용 가능
  공리 L1 (역사) = "이 문서는 1세기 텍스트로서 정합성을 가진다" → 역사학자도 사용 가능
  공리 L2 (정경) = "성경 66권은 상호 일관적으로 설계됐다"     → 기독교인 사용
  공리 L3 (KJV)  = "KJV는 무오한 하나님의 말씀이다"           → 현재 완전 발동
       ↓
  [동일한 파이프라인]
       ↓
  [판결: "공리셋 L{n} 하에서의 결론"]
```

**실제 예시**:
- 갈보리 순교설의 ὅπου → τόπος 연결은 **L1(그리스어 문법 사실)**만으로도 성립한다.
- 즉, 비기독교인 그리스어 학자도 "L1 공리 기준으로 이 논증은 유효하다"고 인정 가능하다.
- KJV 신학 없이도 논증의 핵심이 작동한다.
