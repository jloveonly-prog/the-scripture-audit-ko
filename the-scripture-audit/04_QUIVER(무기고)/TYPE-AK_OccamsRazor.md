# 🏹 SKILL-38: TYPE-AK — 오컴의 면도날 (Occam's Razor / Principle of Parsimony)
**"동일한 증거를 설명하는 가설 중 더 단순한 것이 우선한다"**

> **언제 발동하는가**: 갈보리설과 로마설이 동일한 성경 본문을 설명할 때.
> 두 가설의 설명력이 비슷해 보일 때, 더 단순하고 적은 가정을 요구하는 가설을 선택.
>
> **한 줄 정의**: *"필요 없는 가정을 제거하라. 더 단순한 설명이 더 진실에 가깝다."*

---

## 발동 조건
`
1. 두 가설이 동일한 증거를 설명하는 것처럼 보일 때
2. 상대 가설이 추가적 가정(외부 전통, 역사적 배경)을 요구할 때
3. 성경 내부에서만 작동하는 단순한 설명이 존재할 때
`

## 실행 단계
`
[STEP 1] 가설별 가정 목록 작성
  → 각 가설이 참이려면 필요한 전제/가정을 나열한다.

[STEP 2] 가정 수 비교
  → 어느 가설이 더 적은 가정으로 증거를 설명하는가?

[STEP 3] 단순성 판정
  → 더 적은 가정 = 더 단순한 설명 = 오컴의 면도날 기준 우선

[STEP 4] 판결
  → "더 단순한 가설을 채택한다."
`

## 📌 레퍼런스 케이스: 갈보리설 vs 로마설

| 가정 항목 | 갈보리설 | 로마설 |
|:---|:---:|:---:|
| 요 13:36 ὅπου = 골고다 (성경 내부) | ✅ 필요 | ❌ 불필요 |
| 베드로가 로마에 갔다는 성경 근거 | ❌ 불필요 | ⚠️ 필요 (but 없음) |
| AD 180년 이후 교부 전통 채택 | ❌ 불필요 | ✅ 필요 |
| 성경 외부 역사 자료 수용 | ❌ 불필요 | ✅ 필요 |
| 성경 내부 일관성만으로 완결 | ✅ 완결 | ❌ 미완결 |

> **TYPE-AK 판결**: 갈보리설은 성경 내부만으로 완결되는 단순한 설명이다.
> 로마설은 성경 외부 전통을 추가로 채택해야 작동한다.
> 오컴의 면도날 기준 — 갈보리설 우선. ✅ CONFIRMED
