# 🔬 SKILL-05: TYPE-E — 경쟁 모델 전수 기각 파이프라인

> **언제 발동하는가**: 기존에 여러 해석이 경쟁하는 난제, 또는 "필사 오류"로 처리하려는 시도가 있을 때.
> **핵심 의심**: "지금 제시되는 해석 외에 다른 가능성이 있는가? 그것은 왜 기각되는가?"

---

## 실행 단계

```
[STEP 1] 경쟁 모델 전체 목록화
  → 해당 난제를 설명하려는 모든 해석 모델을 나열한다 (최소 3개 이상).
  → | # | 모델명 | 핵심 주장 |

[STEP 2] 각 모델에 치명적 한계 탐지 (5대 검증 항목 전수 적용)
  각 모델에 대해 다음을 확인한다:
  □ 텍스트 보존성: 단 하나의 성경 구절도 수정/삭제하지 않고 설명 가능한가?
  □ 전수 정합성: 관련 모든 기록과 동시에 일치하는가?
  □ 생물학적 가능성: 나이·기간이 현실적 수치 범위 내인가?
  □ 사건 공백(Gap): 필요한 물리적 행동을 할 수 있는 시간/공간 공백이 존재하는가?
  □ 수학적 정합: 역산했을 때 제3의 성경 수치와 일치하는가?

[STEP 3] 기각 표 작성
  → | 경쟁 모델 | 치명적 한계 | 생존 여부 |
  → 치명적 한계가 단 하나라도 있으면 ❌ 기각.

[STEP 4] 생존 모델 채택 및 전수 교차 검증
  → 생존한 모델을 채택하고, 관련 모든 구절을 대입하여 전수 정합을 최종 확인한다.
```

## 📌 레퍼런스 케이스: 십자가 처형 요일 검증

```
[경쟁 모델 목록]:

  모델 1: 금요일 십자가설
  - 핵심 주장: 예수님이 금요일에 처형됨
  - 치명적 한계: 금~일요일 = 최대 36~40시간 → 마 12:40 "3일 3밤" (72시간) 불충족 ❌

  모델 2: 목요일 십자가설
  - 핵심 주장: 목요일 처형 → 금요일 큰 안식일 → 토요일 정기 안식일
  - 치명적 한계: 두 안식일 사이 평일(금요일)이 없어짐
    → 막 16:1 "안식일이 지난 후 향품 구매" 기록과 충돌 (향품 살 평일이 사라짐) ❌

  모델 3: 수요일 십자가설
  - 핵심 주장: 수요일 처형 → 목요일 큰 안식일 → 금요일 향품 구매 → 토요일 정기 안식일 → 일요일 부활
  - 텍스트 보존성: 모든 구절 수정 없이 설명 ✅
  - 전수 정합성: 막 16:1, 눅 23:56, 마 12:40 동시 충족 ✅
  - 수학적 정합: 수~토 = 정확히 72시간 (3일 3밤) ✅

[타임라인 전수 검증]:
  수요일: 유월절 식사 후 처형, 장사
  목요일: 큰 안식일 (유월절 특별 안식)
  금요일: 향품 구매 공백 (막 16:1 충족)
  토요일: 정기 안식일에 쉼 (눅 23:56 충족)
  일요일 새벽: 부활 (마 12:40 72시간 충족) ✅

[채택]: 수요일 십자가설 — 유일하게 5대 검증 모두 통과한 생존 모델
```

## 📌 레퍼런스 케이스 2: 베드로 순교 장소 경쟁 모델 기각

```
[경쟁 모델 목록]:

  모델 1: 로마 순교설
  - 핵심 주장: 초대교회 전승에 따라 로마에서 순교
  - 치명적 한계: 성경에 베드로의 로마 방문 기록 전무. 전승은 AD 180년 이후 등장 ❌

  모델 2: 바빌론 순교설
  - 핵심 주장: 벧전 5:13의 바빌론에서 사역 중 순교
  - 치명적 한계: 요나 예표(동방→서방 갈보리 회귀) 미완성.
    바빌론에서 멈추면 예표 구조 파괴 ❌ (TYPE-F)

  모델 3: 예루살렘 다른 장소설
  - 핵심 주장: 예루살렘이되 갈보리가 아닌 다른 곳
  - 치명적 한계: θέλω+ὅπου+ἀκολουθήσεις 체인이
    예수님이 죽으신 바로 그 장소를 가리킴 → 다른 장소 불가 ❌ (TYPE-S+G)

  모델 4: 장소 미기록설
  - 핵심 주장: 성경이 장소를 기록하지 않았으므로 알 수 없다
  - 치명적 한계: 요 13:36이 장소 예언을 주었고,
    Narrowing Pressure에 의해 갈보리만 배타성 유지 ❌ (TYPE-N)

[채택]: 갈보리(골고다) 순교설 — 유일한 생존 모델
```
