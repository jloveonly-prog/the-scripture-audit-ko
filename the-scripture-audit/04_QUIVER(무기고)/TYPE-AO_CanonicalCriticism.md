# 🏹 SKILL-42: TYPE-AO — 정경비평 (Canonical Criticism)
**"이 구절은 성경 전체 서사 안에서 어떤 기능을 하는가?"**

> **언제 발동하는가**: 특정 구절이 성경 전체 정경(Canon)의 구조 안에서 차지하는 위치와
> 기능을 분석하여 그 신학적 의미를 확정할 때.
>
> **한 줄 정의**: *"성경의 부분은 전체의 맥락에서 읽어야 그 기능이 드러난다."*

---

## 발동 조건
`
1. 구절의 의미가 단독으로는 불명확하지만 전체 정경 흐름에서 확정될 때
2. 베드로 예언이 요한복음 전체 서사에서 하는 역할을 분석할 때
3. 구약 예언과 신약 성취의 연결이 정경 전체를 통해 작동할 때
`

## 실행 단계
`
[STEP 1] 정경적 위치 확인
  → 분석 구절이 성경 전체에서 어느 위치에 있는가?

[STEP 2] 서사적 기능 분석
  → 이 구절이 성경 전체 서사에서 어떤 역할을 하는가?
  → 예언 / 성취 / 전환점 / 결론 / 포문

[STEP 3] 연결 구조 확인
  → 이 구절이 다른 정경 책들과 어떻게 연결되는가?

[STEP 4] 정경적 의미 선언
  → "이 구절은 성경 전체 서사의 [기능]을 담당한다."
`

## 📌 레퍼런스 케이스: 요한복음 내 베드로 예언의 정경적 기능

| 구절 | 정경적 기능 |
|:---|:---|
| 요 13:36 | 베드로 서사의 **예언적 포문(開門)** — "장래에 따라올 것" |
| 요 18:15-27 | 부인 — 예언의 **첫 번째 실패 지점** |
| 요 21:15-19 | 회복 + 재예언 — **예언의 확정과 명령으로 전환** |
| 요 21:19b | ἀκολούθει — **예언의 성취 명령** (수미상관 닫힘) |

> **TYPE-AO 판결**: 요 13:36의 ὅπου 예언은 정경적으로 요 21:19의 명령으로 완결되는 서사 단위다.
> 이 구조에서 ὅπου의 지시 대상이 "어딘가"로 남는 것은 정경적 서사 완결을 파괴한다.
> ∴ ὅπου = 골고다가 정경적으로 유일한 완결이다. ✅ CONFIRMED
