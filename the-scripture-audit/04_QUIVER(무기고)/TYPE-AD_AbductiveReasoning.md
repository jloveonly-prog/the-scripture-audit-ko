# 🏹 SKILL-31: TYPE-AD — 귀추법 (Abductive Reasoning / Inference to the Best Explanation)
**"모든 증거를 가장 완전하게 설명하는 가설이 참이다"**

> **언제 발동하는가**: 여러 경쟁 가설 중 어느 것이 진실인지를 확정할 때.
> 모든 수집된 증거를 놓고, **각 가설이 그 증거들을 얼마나 잘 설명하는지** 비교 평가하여
> 가장 완전한 설명력을 가진 가설을 채택한다.
>
> **한 줄 정의**: *"어느 가설이 모든 증거를 가장 단순하고 완전하게 설명하는가?"*

> [!IMPORTANT]
> **TYPE-E(경쟁 모델 기각)와의 차이:**
> - TYPE-E = 경쟁 가설 하나하나를 개별적으로 기각
> - TYPE-AD = 모든 가설을 **증거 설명력 기준으로 동시 비교** → 최선 가설 채택
> TYPE-E는 소거(제거) / TYPE-AD는 선택(채택).

---

## 발동 조건

`
다음 상황에서 TYPE-AD를 발동한다:
1. "갈보리설과 로마설 중 어느 쪽이 더 성경적인가?"처럼 복수 가설의 우열을 가릴 때
2. 소거법(TYPE-N)으로 대안들을 제거한 후, 남은 가설의 채택 근거를 강화할 때
3. 상대가 "가장 강력한 해석일 뿐이지 확정은 아니다"라고 주장할 때
   → 귀추법은 "가장 강력한 설명 = 채택해야 할 가설"임을 공식화
`

---

## 실행 단계

`
[STEP 1] 증거 목록 확정 (Evidence Inventory)
  → 본문에서 확보된 모든 증거를 나열한다.
  → (예: ὅπου 장소 부사, 요 21:19 ἀκολουθέω, σημαίνων 공식, σύ 강조 대명사...)

[STEP 2] 경쟁 가설 목록 (Competing Hypotheses)
  → 검증 대상 가설들을 명시한다.
  → (예: 갈보리설, 로마설, 순교 일반설, 천국설)

[STEP 3] 설명력 매트릭스 구성 (Explanatory Matrix)
  → 각 가설이 각 증거를 설명하는가? Yes(✅) / No(❌) / Partial(⚠️)

[STEP 4] 최선 가설 채택 (Best Explanation Adoption)
  → 모든 증거를 ✅로 설명하는 가설 = 귀추법 판결 대상
  → 채택 기준: 설명력 완전성 + 단순성 (오컴의 면도날 보조 적용)

[STEP 5] 판결 선언 (Verdict)
  → "가설 X는 모든 증거를 설명하는 유일한 가설이다. ∴ X를 채택한다."
`
