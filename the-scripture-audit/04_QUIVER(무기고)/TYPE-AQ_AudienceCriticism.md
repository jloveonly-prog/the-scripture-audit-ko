# 🏹 SKILL-44: TYPE-AQ — 청중비평 (Audience Criticism)
**"원래 청중은 이것을 어떻게 이해했는가?"**

> **언제 발동하는가**: 1세기 유대인·제자들이 특정 발화나 구절을 들었을 때
> 어떻게 이해했을지를 분석하여 현대적 오독을 차단할 때.
>
> **한 줄 정의**: *"원래 청중의 귀로 들어야 원래 의미가 들린다."*

---

## 발동 조건
`
[범용]
1. 현대적 해석이 1세기 문화·언어·문맥과 충돌할 때
2. 원래 청중(유대인, 헬라인, 초기 교회)이 특정 발화를 어떻게 이해했는지 확정할 때
3. 성경 개념을 현대적 개념으로 오역하는 것을 차단할 때
   (예: 고대 유대 결혼 관습, 정결례, 성전 제도, 안식일 규례 등)
4. "당시 청중에게 이 발화는 어떻게 들렸는가?"가 논쟁의 핵심일 때

[전형적 코드] 아래 문장 감지 시 즉시 발동:
  - "당시 문화에서 [개념X]는 [Y]를 의미했다"
  - "1세기 유대인들에게 [발화Z]는 [의미]로 들렸다"
  - "현대인의 관점으로 해석하면 오해가 생긴다"
`

## 실행 단계
`
[STEP 1] 원래 청중 확정
  → 이 발화/기록의 1차 청중은 누구인가?
  → (예: 최후 만찬 담화 = 열두 제자)

[STEP 2] 1세기 문화·언어 맥락 복원
  → 원래 청중이 이 단어/개념을 어떻게 이해했을 문화적 배경

[STEP 3] 원래 이해 복원
  → 1차 청중의 귀에 이 발화는 어떻게 들렸는가?

[STEP 4] 현대적 오독 기각
  → 현대적 해석이 1차 청중의 이해와 다르다면 — 현대적 해석이 오독이다.
`

## 📌 레퍼런스 케이스: 요 13:36 최후 만찬 청중 분석

**청중**: 베드로 포함 열두 제자 — 1세기 유대 어부·세리 등

| 질문 | 1세기 청중의 이해 |
|:---|:---|
| "내가 가는 곳(ὅπου)"을 들었을 때 | 물리적 장소 — "어디 가시는 겁니까?" (13:36a 베드로 질문) |
| "나중에 따라올 것"을 들었을 때 | 물리적 동행의 미래 약속 |
| ἀκολουθέω를 들었을 때 | 스승을 몸으로 따라가는 제자 도리 (1세기 랍비 전통) |
| "장래에"를 들었을 때 | 시간적 지연 — "지금은 못 하지만 나중에 할 것" |

> 1세기 유대 제자들에게 ὅπου = 천국이라는 해석은 발생하지 않았다.
> 베드로 자신이 13:36a에서 "어디로 가십니까?"라고 물었다 — **물리적 장소 질문**.

> **TYPE-AQ 판결**: 원래 청중(제자들)은 ὅπου를 물리적 장소로 이해했다.
> 이는 13:36a 베드로의 질문으로 텍스트 내에서 직접 확인된다.
> 현대적 "천국" 해석은 1차 청중의 이해를 벗어난 오독이다. ✅ CONFIRMED
