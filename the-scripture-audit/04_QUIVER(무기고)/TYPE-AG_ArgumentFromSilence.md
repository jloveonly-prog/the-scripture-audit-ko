# 🏹 SKILL-34: TYPE-AG — 침묵 논증 (Argument from Silence / Argumentum ex Silentio)
**"성경이 침묵한다는 것 자체가 증거다"**

> **언제 발동하는가**: 상대의 주장이 성경 내에서 **완전히 침묵**할 때.
> 성경의 침묵은 "몰랐다"가 아니라 "기록할 필요가 없었다" 또는 "그 사건이 없었다"는 뜻일 수 있다.
>
> **한 줄 정의**: *"성경이 전혀 언급하지 않는 것을 성경적 진리라 부를 수 없다."*

---

## 발동 조건
`
[범용]
1. 상대의 주장이 성경 어디에도 나오지 않을 때
2. 어떤 교리/사건/인물이 성경에서 완전히 침묵할 때
   (예: 특정 인물의 이동 경로, 특정 장소에서의 사건, 미언급 교리)
3. "성경에 없다 = 성경이 지지하지 않는다"를 공식화할 때
4. 외부 전통/추정/해석이 성경 내부 기록을 우선하려 할 때

[전형적 코드] 아래 문장 중 하나 이상 감지 시 즉시 발동:
  - "[주장]은 성경에 기록이 없다"
  - "[교리/인물/사건]...에 대한 성경의 언급이 없다"
  - "[AD 180년 이후, 400년 이후...] 전승이다"
`

## 실행 단계
`
[STEP 1] 침묵 범위 확정
  → 상대 주장이 성경 내에서 언급된 구절: 0개 또는 N개 확인
  → 관련 성경 책(사도행전, 서신서, 복음서 등) 전수 스캔

[STEP 2] 침묵의 의미 분석
  → 성경이 해당 주장을 기록할 기회가 있었는가?
  → 그 기회에도 기록하지 않았다면 → 의도적 침묵 (부재의 증거)

[STEP 3] 침묵 vs. 근거 대비
  → 우리 주장: 성경 내부 구조적 근거 존재 ✅ / ❌
  → 상대 주장: 성경의 완전한 침묵

[STEP 4] 판결
  → 성경이 완전히 침묵하는 주장은 성경적 근거가 없다.
  → 성경 내부 근거가 있는 주장이 성경적으로 우선한다.
`

## 📌 로마설의 성경적 침묵 지도

| 성경 책 | 베드로 언급 | 로마 순교 언급 |
|:---|:---:|:---:|
| 사도행전 | ✅ 다수 | ❌ 전무 |
| 베드로전서 | ✅ 저자 | ❌ 전무 |
| 베드로후서 | ✅ 저자 + 순교 암시 | ❌ 로마 언급 없음 |
| 요한복음 | ✅ 순교 예언 | ❌ 로마 언급 없음 |
| 로마서 | ✅ 바울이 로마 성도에게 | ❌ 베드로 로마 체류 언급 없음 |

> **TYPE-AG 판결**: 로마설은 성경의 완전한 침묵 속에 인간 전통이 채워진 것이다.
> 성경 내부 논증이 존재하는 갈보리설이 성경적으로 유일한 근거를 가진다. ✅ CONFIRMED
