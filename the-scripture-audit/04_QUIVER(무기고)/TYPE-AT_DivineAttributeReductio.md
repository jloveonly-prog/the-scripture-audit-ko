# 🔥 SKILL-47: TYPE-AT — 신성 속성 귀류법 (Divine Attribute Reductio)
**"온갖 좋은 선물과 완전한 은사는 위로부터 내려오나니 곧 빛들의 아버지께로부터라" — James 1:17 KJV**

> **언제 발동하는가**: "하나님이 X를 직접 창조/행하셨다"는 주장이 제기될 때,
> AND 그 X의 속성이 하나님의 **본질적 속성**(거룩, 선, 빛, 진리)과 **정면 충돌**할 때.
> **핵심 질문**: *"거룩하신 하나님이 이것을 직접 창조하셨다면, 하나님의 어떤 속성이 훼손되는가?"*

> [!IMPORTANT]
> **TYPE-AC/TYPE-AI와 혼동 금지:**
> - `TYPE-AC` = 역가설(¬A)을 세워 **텍스트 간 모순**을 적발
> - `TYPE-AI` = 상대 가설을 성경에 대입 → **논리적 모순** 폭발
> - `TYPE-AT` = 하나님의 **본질적 속성** vs 행위 **결과**의 충돌 적발
> TYPE-AC/AI는 논리적 파괴 / TYPE-AT는 **신성 모독 수준의 파괴** (감정적+논리적 이중 구조).

---

## ⚙️ 발동 조건

```
□ "하나님이 X를 직접 창조/행하셨다"는 주장이 제기될 때
□ AND 그 X의 속성이 하나님의 본질적 속성(거룩, 선, 빛, 진리)과 정면 충돌할 때
□ "하나님이 악을 창조하셨는가?" 계열의 질문이 쟁점일 때
□ "하나님이 왜 더러운 영/사악한 영을 만드셨는가?"에 대한 응답이 필요할 때
```

---

## ⚙️ 실행 단계

```
[STEP 1] 쟁점 행위/결과를 명시
  → "하나님이 X를 직접 창조(Bara)하셨다"
  → X의 정체를 KJV에서 확인한다.
  → | X | KJV 명칭 | 구절 | 속성 |

[STEP 2] X의 속성을 KJV에서 확인
  → X가 가진 속성(더러움, 사악함, 결함 등)을 KJV 용어로 특정한다.
  → | 대상 | KJV 명칭 | 속성 키워드 |

[STEP 3] 하나님의 본질적 속성을 KJV 앵커로 고정
  → 하나님의 속성 중 X의 속성과 정면 충돌하는 것을 최소 3개 이상 수집한다.
  → 【 하나님의 속성 앵커 목록 (표준) 】
     ① 야 1:17 — "온갖 좋은 선물... 빛들의 아버지" (선+빛)
     ② 합 1:13 — "눈이 정결하시므로 악을 보실 수 없으시며" (정결+악 불용)
     ③ 요일 1:5 — "하나님은 빛이시라 그 안에 어둠이 전혀 없으시다" (빛+어둠 부재)
     ④ 신 32:4 — "그의 행하신 일은 완전하고" (완전)
     ⑤ 사 6:3 — "거룩하다, 거룩하다, 거룩하다" (절대 거룩)

[STEP 4] 속성 충돌 판정표 작성
  → | X | X의 속성 | 하나님의 속성 | 충돌? | KJV 앵커 |
  → 충돌이 발생하면 💥 표시.
  → 충돌 = "하나님이 X를 직접 Bara하셨다"는 주장이 신성 속성을 훼손

[STEP 5] 대안 요구 → 유일 대안 제시
  → "X의 기원을 하나님의 속성을 훼손하지 않고 설명하라"
  → 유일 대안: 타락한 피조물의 파생물 (하나님이 아닌 존재에서 기원)
  → 하나님은 선한 것(아들들)을 창조하셨고, 아들들이 타락하여 더러운 파생물이 됨
  → 하나님의 창조 = 선 → 피조물의 타락 = 악 → 하나님의 속성 완전 보전
```

## 📌 레퍼런스 케이스: 더러운 영을 하나님이 창조하셨는가?

```
[STEP 1] 쟁점:
  → "하나님이 마귀(더러운 영, 점치는 영, 벙어리 영)를 직접 창조(Bara)하셨다"

[STEP 2] X의 속성 확인:

  | 마귀 종류 | KJV 명칭 | 속성 키워드 | 구절 |
  |:---|:---|:---:|:---|
  | 더러운 영 | unclean spirit | 불결(unclean) | 막 1:23 |
  | 점치는 영 | spirit of divination | 마술(divination) | 행 16:16 |
  | 벙어리 영 | dumb spirit | 결함(dumb) | 막 9:25 |
  | 사악한 영 | evil spirit | 악(evil) | 삼상 16:14 |

[STEP 3] 하나님의 속성 앵커:
  → 야 1:17 — "온갖 좋은 선물과 완전한 은사는 빛들의 아버지께로부터"
  → 합 1:13 — "눈이 정결하시므로 악을 보실 수 없으시며"
  → 요일 1:5 — "하나님은 빛이시라 그 안에 어둠이 전혀 없으시다"

[STEP 4] 속성 충돌 판정:

  | X | X의 속성 | 하나님의 속성 | 충돌? | KJV 앵커 |
  |:---|:---:|:---:|:---:|:---|
  | 더러운 영(unclean) | 불결 | 정결 | 💥 | 합 1:13 |
  | 점치는 영(divination) | 마술 | 가증함 선언 | 💥 | 신 18:10-12 |
  | 벙어리 영(dumb) | 결함 | 완전 | 💥 | 신 32:4 + 출 4:11 |
  | 사악한 영(evil) | 악 | 선·빛 | 💥 | 야 1:17 + 요일 1:5 |

  → 4/4 전부 충돌 = "하나님이 이들을 직접 Bara하셨다"는 주장은
    하나님의 정결·선·완전·빛 속성을 4중으로 훼손

[STEP 5] 대안:
  → 유일한 속성 보전 설명:
    하나님이 선한 존재(하나님의 아들들)를 창조(Bara)하셨고,
    그 아들들이 자유의지로 타락하여 자신들의 영적 에너지를 분할·파생시킨
    민족들의 영이 마귀가 된 것.
  → 하나님의 창조 = 선한 아들들
  → 아들들의 타락 = 더러운 파생물(마귀)
  → ∴ 하나님의 속성 완전 보전 ✅
```

## ⚠️ 주의사항

```
□ 이 무기는 감정적 파괴력이 매우 높다.
  → "당신은 하나님이 더러운 영을 만드셨다고 말하는 것입니까?"
  → 이 질문 자체가 상대방을 방어 불능 상태로 몰아넣는다.
□ 그러나 감정적 공격만으로는 안 된다 — 반드시 KJV 앵커 3개 이상으로 뒷받침할 것.
□ 삼상 16:14 "주로부터 온 사악한 영(evil spirit from the LORD)" 반박 준비 필수:
  → 하나님이 사악한 영을 "보내셨다(sent)"는 것은 "창조(Bara)하셨다"와 다르다.
  → 하나님은 이미 존재하는 사악한 영을 심판의 도구로 "사용(deploy)"하신 것.
  → 비유: 판사가 사형집행인을 보내는 것 ≠ 판사가 사형집행인을 만든 것.
```
