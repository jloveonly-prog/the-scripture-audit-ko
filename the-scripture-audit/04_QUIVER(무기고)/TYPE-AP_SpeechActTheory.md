# 🏹 SKILL-43: TYPE-AP — 화행이론 (Speech Act Theory)
**"발화는 단순한 정보가 아니다 — 약속·명령·예언·선언이다"**

> **언제 발동하는가**: 발화의 언어적 형태가 아니라 **발화 행위의 기능**(약속인가, 명령인가, 예언인가)을
> 분석하여 정확한 의미를 확정할 때.
>
> **한 줄 정의**: *"무슨 말인가가 아니라, 무슨 행위를 하는 말인가를 분석한다."*

> [!NOTE]
> **Austin·Searle의 화행이론 3구분:**
> - **발화행위(Locution)**: 문자적으로 말해진 것
> - **발화수반행위(Illocution)**: 발화로 수행되는 행위 (약속/명령/경고/예언)
> - **발화효과행위(Perlocution)**: 발화가 청자에게 미치는 효과

---

## 발동 조건
`
[범용]
1. 발화의 언어적 형태와 실제 기능(약속/명령/예언/경고)이 다를 때
2. "이 말씀은 단순한 권유/위로/설명이다"라는 반론을 기각할 때
3. 하나님/예수님의 발화가 어떤 화행을 수행하는지 확정할 때
4. 선지자/사도의 발화가 예언인지, 명령인지, 선언인지 구분이 필요할 때

[전형적 코드] 아래 문장 감지 시 즉시 발동:
  - "이것은 단순한 [격려/위로/조언]이다"
  - "이 명령은 영적 의미이지 물리적 명령이 아니다"
  - "[하나님/예수님/사도]의 발화가 약속인지, 예언인지, 명령인지 다툼이 있을 때"
`

## 실행 단계
`
[STEP 1] 발화 텍스트 확정
  → 분석 대상 발화를 원어로 확정한다.

[STEP 2] 발화수반행위 분류
  → 이 발화는 무엇을 수행하는가?
     약속(Promise) / 명령(Command) / 예언(Prophecy) / 경고(Warning) / 선언(Declaration)

[STEP 3] 문맥 적용
  → 화자·청자·상황의 맥락에서 발화행위를 최종 확정한다.

[STEP 4] 의미 선언
  → "이 발화의 발화수반행위는 [기능]이다."
`

## 📌 레퍼런스 케이스

**요 21:18-19 분석:**

| 발화 | 문자적 내용 | 발화수반행위 | 근거 |
|:---|:---|:---:|:---|
| 21:18 "다른 사람이 띠를 띠우고 끌고 가리라" | 미래 사건 묘사 | **예언(Prophecy)** | σημαίνων 표지(요 12:33 = 십자가형) |
| 21:19 "이 말씀은 베드로가 어떠한 죽음으로 하나님께 영광을 돌릴 것을 가리켜" | 저자 해설 | **선언(Declaration)** | 요한이 직접 해석 |
| 21:19b "나를 따르라(ἀκολούθει)" | 명령형 동사 | **명령(Command) + 예언 실행 명령** | 13:36 예언의 이행 촉구 |

> **TYPE-AP 판결**: "나를 따르라"는 단순 권유가 아니라
> 13:36 예언을 실행하는 명령 + 십자가 죽음으로의 동행 요청이다.
> 이 발화수반행위는 물리적 장소(골고다)로의 동행을 전제한다. ✅ CONFIRMED
