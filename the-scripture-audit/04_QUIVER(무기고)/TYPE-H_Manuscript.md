# 🔬 SKILL-08: TYPE-H — 사본학적 증거 독립성 역전 파이프라인

> **언제 발동하는가**: "사본이 많은 쪽이 옳다"는 수량 논리로 공격받을 때.
> **핵심 의심**: "공격자의 사본 5,800개는 하나의 출처에서 파생된 복사본 아닌가? 독립적인 출처의 수는 몇 개인가?"

---

## 실행 단계

```
[STEP 1] 사본 수 공격 인정 후 즉시 전선 이동
  → "사본 수가 열세임을 인정한다. 그러나 사본 수 ≠ 원본 근접성이다."
  → 이유: 사본 수 = 그 텍스트가 많이 복사되었다는 것.
           원본에 가깝다는 증거가 아님.

[STEP 2] 독립 출처 계산
  → 다수 사본들이 단일 출처에서 파생된 것인지 확인
    (예: 모두 알렉산드리아 계열 → 독립 출처 사실상 1개).
  → 지지 사본들의 독립 출처를 분류:
     | 출처 | 지역 | 언어 | 연대 | 독립성 |
  → 같은 지역/계열에서 파생된 것들은 물리적 수와 무관하게 독립 출처 1개로 계산.

[STEP 3] 지역·언어·시대 독립성 가중치 행렬 작성
  → 각 독립 출처가 서로 다른 지역, 언어, 시대에서 동일한 본문을 전승하는지 확인.
  → 핵심: 서로 연락 불가능한 곳에서 동일한 본문 → 강력한 독립 증거.

[STEP 4] 시간 역산: 공격 전선을 "최초 증거의 연대"로 이동
  → 교부 인용 증거가 있다면: 현존 물리적 사본보다 훨씬 이른 시기의 증거.
  → 공식 문서(공의회, 주교 신앙고백서)에 인용된 증거:
    개인이 아닌 교회 공동체의 증언.

[STEP 5] "삭제 패턴"의 일관성 추적
  → 공격하는 사본 계열이 이 구절 외에 다른 구절들도 일관되게 삭제하는 패턴이 있는가?
  → 패턴이 있다면: 해당 사본 계열의 신뢰성 자체를 문제 삼을 수 있다.
```

## 🆕 서브 프로토콜: 본문 내적 구조 증거 역전 (Internal Structural Evidence Reversal) — v1.1

> **발동 조건**: 사본 수 열세를 공격받을 때, 사본 전쟁에서 벗어나 **"본문 자체가 원본임을 증명"**하는 전선으로 이동한다.
> **핵심 원리**: 삽입자는 다른 구절의 문법적 필수 구조까지 동시에 설계하지 않는다. 본문 내 구조적 필연성이 있다면 원본 설계 증거다.

```
[STEP 1] 사본 전선 이탈 선언
  → "사본 수 논쟁은 보조 전선이다. 본문 자체의 내적 증거로 이동한다."

[STEP 2] 내적 구조 증거 발굴
  → 삭제 대상 구절이 인접 구절의 어휘·문법·대칭과 분리 불가능한 관계인지 확인
  → 분리 시 고아 어휘 또는 문법 공백 발생 여부 검증 (TYPE-G 구조적 쌍 잠금과 연계)

[STEP 3] 교부 인용 연대표 작성 — 사본보다 이른 증거
  → 현존 물리적 사본보다 교부 인용이 더 이르면 → 전선 역전
  → 표 형식:
    | 교부 | 연대 | 인용 방식 | 증거 등급 |
  → 증거 등급:
     🟡 간접 = 유사 표현 사용 (scriptum est 없음)
     🟠 강한 간접 = scriptum est + 삼위 명시
     🔴 직접 = 교회 공식 문서에서 해당 구절 명시 인용

[STEP 4] 독립 출처 역전 선언
  → 공격측 사본 수 X개 = 독립 출처 N개 (파생 계열 정리)
  → 방어측 교부 증거 = 독립 출처 M개 (지역·언어·시대 각각 다른 독립 전통)
  → M > N이면 → 독립성 역전 선언
```

## 📌 레퍼런스 케이스: 요한1서 5:7 콤마 구절

```
[공격측 사본 분석]:
  알렉산드리아 파생 사본 5,800개 → 독립 출처 사실상 1개 (알렉산드리아 단일 계열)

[방어측 독립 출처 4개]:
  ① 북아프리카 라틴 전통
     - 터툴리안 (Tertullian, 200AD) — 삼위일체 교리 인용
     - 키프리아누스 (Cyprian, 250AD) — 요일 5:7 직접 인용 확인
     - 독립 출처 1 ✅

  ② 히스파니아 라틴 전통
     - 프리스킬리안 (Priscillian, 380AD) — 스페인 지역에서 인용
     - 독립 출처 2 ✅

  ③ 로마/불가타 계열
     - 히에로니무스(Jerome) 서문 (390AD) — 삭제 문제를 직접 언급하며 원문 보존 주장
     - 독립 출처 3 ✅

  ④ 484AD 카르타고 공의회 공식 신앙고백서
     - 수백 명의 주교가 서명한 공식 교회 문서에 요일 5:7 인용
     - 개인 기록이 아닌 교회 공동체 전체의 증언
     - 독립 출처 4 ✅

[역전 결과]:
  알렉산드리아 5,800개 = 독립 출처 1개
  방어측 = 독립 출처 4개 → 4:1 독립성 역전 ✅

[삭제 패턴 추적]:
  알렉산드리아 계열의 일관된 삭제 목록:
  → 요일 5:7 (삼위일체 구절)
  → 요 5:4 (천사가 물을 동함)
  → 행 8:37 (에디오피아 내시의 신앙고백)
  → 일관된 구조적 삭제 패턴 존재 → 계열 신뢰성 문제 제기 가능 ✅
```
