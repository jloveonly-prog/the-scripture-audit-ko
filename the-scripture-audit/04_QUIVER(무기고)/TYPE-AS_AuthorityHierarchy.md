# ⚡ SKILL-46: TYPE-AS — 권능 위계 분석 (Authority Hierarchy Analysis)
**"The heaven, even the heavens, are the LORD'S: but the earth hath he given to the children of men." — Psalm 115:16 KJV**

> **언제 발동하는가**: "하나님만 하실 수 있는 행위"와 "피조물도 할 수 있는 행위"의 **구분이 쟁점**일 때.
> 성경에서 동일하게 "만들다/창조하다"로 번역된 단어들이 실제로는 **다른 권한 수준**을 가리키는 경우,
> 원어 분리와 주체 자격 전수 검증을 통해 **권능의 위계(Hierarchy)**를 확립한다.
> **핵심 질문**: *"이 행위는 하나님만의 전유물인가, 위임받은 피조물도 수행 가능한가?"*

> [!IMPORTANT]
> **TYPE-G와 혼동 금지:**
> - `TYPE-G` = 문법 구조(접속사, 성·수·격, 동사 태/법) 해부
> - `TYPE-AS` = 행위의 **권한 수준**(독점 vs 위임) 분석
> TYPE-G는 단어의 문법을 파헤치고 / TYPE-AS는 행위의 자격을 심판한다.

---

## ⚙️ 발동 조건

```
□ "피조물이 생명을 만들 수 있는가?" 같은 창조 주권 질문이 제기될 때
□ "하나님이 X를 위임하셨는가?" 같은 권위 위임 질문이 제기될 때
□ 원어에서 동일한 "만들다" 계열 동사가 다른 의미 수준으로 사용될 때
□ 출 8:18-19(마술사 실패)처럼 "위임 없는 시도→실패" 사례가 관찰될 때
□ 하나님의 창조 주권을 훼손하지 않으면서 피조물의 행위를 설명해야 할 때
```

---

## ⚙️ 실행 단계

```
[STEP 1] 행위 원어 분리 — 쟁점이 되는 행위의 원어를 정밀 분리
  → KJV 영어에서 동일하게 번역된 단어들의 원어 차이를 드러낸다.
  → 표 형식:
  → | 원어 | 영어 (KJV) | 정밀 의미 | 행위 주체 자격 | KJV 증거 |

[STEP 2] 주체 자격 전수 검증 — 해당 원어의 주어를 KJV 전체에서 검색
  → 해당 원어 동사가 사용된 모든 구절에서 주어를 전수 수집한다.
  → 주어가 100% 하나님이면 → 하나님 전용(독점)
  → 주어에 사람/천사/기타 존재가 포함되면 → 위임 가능

[STEP 3] "위임 실패" 사례 수집 — 하나님의 위임 없이 시도→실패한 사례
  → 성경에서 피조물이 해당 행위를 시도했으나 실패한 사례를 찾는다.
  → 실패 사례가 있으면 → "위임 없이는 불가능"의 증거
  → 실패 이유가 명시되어 있으면 → "하나님의 손가락(finger of God)" 등 독점 근거

[STEP 4] 권능 위계표 출력
  → | Level | 권능 | 하나님 전용? | 위임 가능? | KJV 증거 | 위임 실패 사례 |
  → 위계를 Level 1(최상위, 독점)부터 Level N(최하위, 자유)까지 구성한다.

[STEP 5] 쟁점 행위의 위계 배치 → 논쟁 해결
  → 쟁점이 된 행위가 어느 Level에 해당하는지 배치한다.
  → Level이 "위임 가능"이면 → 피조물의 행위 가능 + 하나님 주권 보전
  → Level이 "독점"이면 → 피조물의 행위 불가 확정
```

## 📌 레퍼런스 케이스: Bara vs Asah/Yatsar — 첫 세상 민족의 형성

```
[STEP 1] 원어 분리:

  | 원어 | 영어 (KJV) | 정밀 의미 | 주체 자격 |
  |:---:|:---:|:---|:---:|
  | בָּרָא (Bara) | Create | 무(無)에서 물질과 생명의 근원을 이끌어냄 | 오직 하나님 |
  | עָשָׂה (Asah) | Make | 이미 존재하는 재료를 조합하여 새로운 것을 만듦 | 하나님 + 위임 가능 |
  | יָצַר (Yatsar) | Form | 이미 존재하는 재료(진토 등)를 빚어 형태를 부여함 | 하나님 + 위임 가능 |

[STEP 2] 주체 자격 전수 검증:
  → Bara: 창 1:1, 1:21, 1:27, 2:3, 5:1, 5:2, 6:7, 사 40:26, 42:5, 43:1, 45:7, 45:18...
    → 주어 = 100% 하나님 (예외 0건) ✅
  → Asah: 창 1:7, 1:16, 1:25, 1:31, 2:2, 출 20:11...
    → 주어 = 하나님 다수 + 사람도 주어 가능 (출 31:4 "브살렐이 만들더라")
    → 위임 가능 확인 ✅
  → Yatsar: 창 2:7, 2:8, 2:19, 사 44:2...
    → 주어 = 하나님 다수 + 사 29:16 "토기장이(인간)가 빚는다" 용례 존재
    → 위임 가능 확인 ✅

[STEP 3] 위임 실패 사례:
  → 출 8:18-19 — 파라오의 마술사들이 이(lice, 생명체)를 만들려 했으나
    "그들은 할 수 없었더라(they could not)"
    → 마술사 자백: "이것은 하나님의 손가락(finger of God)이니이다"
    → 하나님의 위임 없이 진토→생명 시도 → 실패 = 위임 없이는 불가능 증거

[STEP 4] 권능 위계표:

  | Level | 권능 | 하나님 전용? | 위임 가능? | KJV 증거 | 위임 실패 사례 |
  |:---:|:---|:---:|:---:|:---|:---|
  | 1 | Bara (무에서 창조) | ✅ | ❌ | 창 1:1 주어 100% 하나님 | — |
  | 2 | 생육의 복 (Be fruitful) | ✅ | ❌ | 창 1:22,28 하나님만 선언 | — |
  | 3 | Asah/Yatsar (형성) | 가능 | ✅ | 출 31:4 브살렐 | 출 8:18 마술사 실패 |
  | 4 | Dominion (통치) | 가능 | ✅ | 창 1:28 아담에게 위임 | — |

[STEP 5] 쟁점 행위 배치:
  → 첫 세상 아들들의 행위 = Level 3 (Asah/Yatsar) = 위임 가능 범위 내
  → 하나님의 창조 주권 (Level 1~2) 미훼손
  → 출 8:18-19의 마술사 실패 = 위임(Authorization)의 유무가 핵심
    → 파라오의 마술사 = 불법 시도 → 실패
    → 첫 세상의 아들들 = 합법적 위임 하에 형성 → 가능
```

## 🔗 연계 TYPE

```
TYPE-AS (권능 위계 분석)
    ├─ 원어 분리의 기반 → TYPE-G (KJV 문법 구조 해부)
    ├─ 위임 실패 사례의 역이용 → TYPE-P (역논법/부메랑)
    ├─ 경쟁 모델 기각(출산 vs 형성) → TYPE-E (경쟁 모델 기각)
    ├─ 위계표→체인 연결 → TYPE-L (연쇄 추론)
    └─ 신성 속성 충돌 검증 → TYPE-AT (신성 속성 귀류법)
```

## ⚠️ 주의사항

```
□ "위임 가능" 판정이 곧 "위임이 실제로 일어났다"를 의미하지 않는다.
  → 위임의 실제 발생 여부는 별도 앵커 구절로 입증해야 한다.
  → TYPE-AS는 "가능성"을 확립하고, 실제 발생은 TYPE-L(체인) + TYPE-AD(귀추법)로 확정한다.
□ Level 1(Bara)의 독점성은 절대적이다.
  → 어떤 상황에서도 피조물이 Bara의 주어가 된 사례가 없다.
  → 이 독점성을 훼손하는 해석은 골 1:16과 정면 충돌한다.
□ 출 8:18-19는 이 무기의 가장 강력한 탄약이다.
  → "하나님의 손가락"이라는 자백은 마귀 자신이 위임 없음을 인정한 것.
```
