# 🔬 SKILL-02π: TYPE-B-π — 지각 필터 (Perception Filter / Witness Capacity Limit)

> **언제 발동하는가**: 증인이 사건 현장에 있었고, 물리적으로 보거나 들었으나, 심리·신체·환경적 이유로 정보를 **정상적으로 처리하지 못한** 경우.
> **핵심 의심**: "이 증인은 보지 못한 것인가, 아니면 봤으나 처리 불가 상태였는가?"

> **분류**: TYPE-B의 독립 확장 무기 (TYPE-B와 병행 발동 가능)
> **출처**: `BVCAP_System_Improvement_v2.md` — [B]_무덤사건_부활아침 S등급 전례 도출

---

## 핵심 원리: 목격의 3상태 모델

```
기존 시스템 (2상태):
  상태 1: 봤다 (Witnessed)
  상태 2: 못봤다 (Did Not Witness)

TYPE-B-π 추가 (3상태):
  상태 1: 봤다 → 정상 처리 → 보고 가능
  상태 2: 못봤다 → 처리 없음 → 보고 없음
  상태 3: 봤다 → 처리 불가 (충격/애도/문화적 선입견) → 왜곡된 보고 ← NEW
```

> **핵심 원칙**: **목격(Witness) ≠ 인식(Perception)**
> 눈이 열려 있어도, 귀가 들려 있어도 — 뇌(심리)가 처리를 거부할 수 있다.

---

## 실행 단계

```
[STEP 1] 증인의 심리·신체 상태 확인
  → 사건 당시 증인이 어떤 감정 상태였는가?
  → 충격, 극도의 슬픔, 두려움, 문화적 선입견, 피로, 혼란 상태였는가?
  → 발동 조건 충족 여부 판별

[STEP 2] 입력 vs 출력 불일치 탐지
  → 증인이 실제로 받은 정보 (Input): 천사 메시지, 예수님 음성, 특정 장면
  → 증인이 실제로 보고한 내용 (Output): 도난 의심, 동산지기로 오인 등
  → Input ≠ Output 이면 TYPE-B-π 발동 확정

[STEP 3] 처리 불가 원인 분류
  → SHOCK: 급격한 충격으로 인한 인지 마비
  → GRIEF: 극도의 애도 상태로 인한 인지 왜곡
  → CULTURAL: 문화적 선입견(부활 불가능 전제)으로 인한 정보 거부
  → PHYSICAL: 피로·부상·환경적 제약으로 인한 인지 제한
  → DIVINE: 초자연적 차단 (눅 24:16 "눈이 가리워져")

[STEP 4] 왜곡 보고의 내부 일관성 검증
  → 왜곡된 보고가 증인의 처리 불가 상태와 논리적으로 일관되는가?
  → 일관됨 → 지각 필터로 완전히 설명 가능 ✅
  → 불일관 → 다른 원인 추가 탐색

[STEP 5] 원어로 지각 상태 확인
  → 관련 동사의 시제·태를 분석: 능동(보다) vs 수동(보이다)
  → 눅 24:16: "ἐκρατοῦντο" (수동태 미완료) = "계속하여 억제되어 있었다"
  → 헬라어 수동태 → 외부적(신적) 차단의 명백한 증거
```

## 적용 사례

### 사례 1: 막달라 마리아의 왜곡된 보고 (충격 + GRIEF)

```
[Input]:  T1-B 시점: 천사가 "어찌하여 우느냐, 그는 여기 계시지 않다"고 말함
[State]:  극도의 애도 상태 (눅 23:55-56, 요 20:11 "무덤 밖에서 울더니")
[Output]: 도난 가설로 보고 (요 20:2 "누가 주를 옮겨다가 어디 두었는지 모르겠다")

→ 판정: TYPE-B-π (GRIEF) — 천사의 말을 들었으나 애도 상태로 인해
          '부활' 해석이 불가능 → '도난' 해석으로 왜곡 출력 ✅
```

### 사례 2: 막달라의 예수님 오인 (충격 + GRIEF)

```
[Input]:  부활하신 예수님이 뒤에 서 계심 (요 20:14)
[State]:  울고 있는 상태 / 부활이 전제되지 않은 인지 프레임
[Output]: 동산지기로 오인 (요 20:15)

→ 판정: TYPE-B-π (GRIEF + CULTURAL) — 물리적으로 목격했으나
          "죽은 자가 살아 있다"는 전제가 없어 인식 불가 ✅
```

### 사례 3: 엠마오 제자들 (신적 차단)

```
[Input]:  부활하신 예수님과 동행하며 대화 (눅 24:13-31)
[State]:  눅 24:16 "그들의 눈이 가리워져 그인 줄 알아보지 못하더라"
[Output]: 낯선 나그네로 인식

→ 판정: TYPE-B-π (DIVINE) — ἐκρατοῦντο (수동태) = 신적 외부 개입으로
          지각이 능동적으로 차단됨. 인간적 감정 문제가 아님 ✅
```
