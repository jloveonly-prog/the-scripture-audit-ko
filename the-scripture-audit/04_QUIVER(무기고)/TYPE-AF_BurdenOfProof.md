# 🏹 SKILL-33: TYPE-AF — 입증 책임 전가 (Burden of Proof Transfer)
**"공격하는 자가 먼저 증명하라"**

> **언제 발동하는가**: 상대가 "갈보리설은 증명이 부족하다"고 공격할 때.
> 공격자에게 입증 책임을 돌려, 성경에 없는 주장(로마설)이 먼저 증거를 제시해야 함을 확립.
>
> **한 줄 정의**: *"없다는 주장이 증거 없이 있다는 주장을 기각할 수는 없다."*

> [!IMPORTANT]
> **발동 원칙**: 기존의 성경 내부 논증(갈보리설)이 존재하는 상황에서,
> 외부 전통만으로 이를 기각하려는 자는 **성경 내부 반증을 먼저 제시해야 한다.**

---

## 발동 조건
`
[범용]
1. 상대가 증거 없이 우리 주장을 기각하려 할 때
2. 성경 내부 논증이 있는 우리 주장에 대해 외부 전통/권위로만 반박할 때
3. "입증 책임은 공격하는 자에게 있다"를 공식화할 때

[전형적 코드] 아래 문장 감지 시 즉시 발동:
  - "[우리 주장]은 명시가 아니다" (근거 없는 기각 시도)
  - "[외부 전통/교부/학계]에서 다르게 본다"
  - 상대가 증거 없이 전통의 권위만으로 반론할 때
  → 반박: "성경 내부 근거가 있는 주장을 기각하려면 성경 내부 반증을 먼저 제시하라"
`

## 실행 단계
`
[STEP 1] 입증 책임 위치 확인
  → 현재 누가 무엇을 주장하는가?
  → 갈보리설: 성경 내부 근거 제시 ✅
  → 로마설: 성경 내부 근거 없음 ❌

[STEP 2] 책임 전가 선언
  → "성경 내부 근거가 있는 갈보리설을 기각하려면,
     성경 내부에서 갈보리설을 반박하는 증거를 제시하라."

[STEP 3] 전통 증거 기각
  → 외부 전통(AD 180년 이후)은 성경 내부 논증을 기각할 수 없음.
  → 성경 = 최고 권위 / 전통 = 2차 자료

[STEP 4] 판결
  → 로마설은 성경 내부 반증 없이 갈보리설을 기각 불가.
  → ∴ 갈보리설 유지. ✅
`

| 주장 | 근거 | 입증 책임 |
|:---|:---|:---:|
| 갈보리설 | 요 13:36 + 19:17 성경 내부 논증 | ✅ 제시 완료 |
| 로마설 | AD 180년 이후 교부 전통 | ❌ 성경 내부 반증 미제시 |

> **TYPE-AF 판결**: 로마설은 성경 내부 반증을 제시하지 못한 채 외부 전통만으로 갈보리설을 공격한다.
> 입증 책임 원칙상, 성경 내부 근거를 가진 갈보리설이 우선한다. ✅ CONFIRMED
