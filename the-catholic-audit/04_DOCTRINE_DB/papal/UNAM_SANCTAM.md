# UNAM-SANCTAM — 교황 복종이 구원에 절대 필요

| 항목 | 내용 |
|:---|:---|
| **ID** | UNAM-SANCTAM |
| **출처** | 교황 보니파시오 8세, 교서 Unam Sanctam |
| **조항** | 결론부 (교의적 정의) |
| **교의 등급** | De Fide |
| **파문(Anathema)** | ❌ 아니오 (직접적 파문 조항 없으나, De Fide 교의로 선언) |
| **연도** | 1302 |

## 원문 (요약)
> "더 나아가 우리는 로마 교황에 대한 복종이 모든 인간 피조물에게 구원을 위해 절대적으로 필요하다고(omnino esse de necessitate salutis) 선언하고 확정하고 정의한다."

## 라틴어 원문
> "Porro subesse Romano Pontifici omni humanae creaturae declaramus, dicimus, definimus et pronuntiamus omnino esse de necessitate salutis."

## 태그
`교황론`, `교회론`, `구원론`, `수위권`, `de_fide`, `절대`, `필수`

## 주장 (Claims)
1. 로마 교황에 대한 복종(subesse)이 모든 인간 피조물에게 구원을 위해 절대적으로 필요하다
2. 하나이고 거룩한 가톨릭 사도 교회 밖에는 구원도, 죄의 사함도 없다 (Extra Ecclesiam nulla salus)
3. 교회에는 영적 검(gladius spiritualis)과 세속적 검(gladius temporalis) 두 자루가 있으며, 둘 다 교회의 권능에 속한다
4. 세속 권력은 영적 권력에 종속된다

## 부정 (Negates)
1. 교황에 대한 복종 없이도 구원받을 수 있다는 명제
2. 가톨릭 교회 밖에서 구원이 가능하다는 명제
3. 세속 권력이 영적 권력과 독립적이라는 명제
4. 교황의 수위권이 영적 영역에만 제한된다는 명제

## 관련 충돌
- → LG-16 (그리스도/교회를 모르는 이들의 구원 가능성)
- → DIGNITATIS-HUMANAE (종교 자유의 권리)
- → NOSTRA-AETATE (타종교에 대한 존중)
- → CCC-847 (자기 잘못 없이 교회를 모르는 이들의 구원)
