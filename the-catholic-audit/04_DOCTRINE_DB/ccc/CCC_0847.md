# CCC-0847 — 교회 밖 구원 가능성

| 항목 | 내용 |
|:---|:---|
| **ID** | CCC-0847 |
| **출처** | 가톨릭 교회 교리서 (Catechism of the Catholic Church) |
| **조항** | §847 |
| **교의 등급** | Sententia Certa |
| **파문(Anathema)** | 아니오 |
| **연도** | 1992 |

## 원문 (요약)
> 그리스도와 그의 교회에 대해 알지 못하면서도 자기 잘못 없이 복음과 교회를 모르는 사람들이 진실한 마음으로 하느님을 찾고, 양심의 명령을 통해 인식되는 하느님의 뜻을 은총의 영향 아래 행동으로 실천하려고 노력하는 경우 영원한 구원을 얻을 수 있다(제2차 바티칸 공의회, 교회헌장 Lumen Gentium §16).

## 태그
`구원론`, `교회론`, `sententia_certa`, `조건부`, `사목적`

## 주장 (Claims)
1. 자기 잘못 없이 그리스도와 교회를 모르는 자도 구원받을 수 있다
2. 구원의 조건은 진실한 마음으로 하느님을 찾는 것이다
3. 양심의 명령에 따라 하느님의 뜻을 실천하려는 노력이 필요하다
4. 이러한 구원은 은총의 영향 아래 이루어진다
5. 교회의 가시적 경계 밖에서도 구원의 가능성이 존재한다

## 부정 (Negates)
1. 교회 밖에는 어떤 구원도 절대적으로 불가능하다 (Extra Ecclesiam nulla salus의 절대적 해석)
2. 세례를 받지 않으면 어떤 경우에도 구원될 수 없다
3. 복음을 듣지 못한 자는 모두 저주받는다
4. 구원은 오직 가톨릭 교회의 가시적 구성원에게만 한정된다
