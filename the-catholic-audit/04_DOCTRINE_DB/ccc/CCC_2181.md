# CCC-2181 — 주일 미사 의무

| 항목 | 내용 |
|:---|:---|
| **ID** | CCC-2181 |
| **출처** | 가톨릭 교회 교리서 (Catechism of the Catholic Church) |
| **조항** | §2181 |
| **교의 등급** | Sententia Certa |
| **파문(Anathema)** | 아니오 |
| **연도** | 1992 |

## 원문 (요약)
> 주일과 의무 축일에 거행되는 성찬례(미사)는 온 교회의 기초이다. 신자들은 중대한 이유(예: 질병, 영아 돌봄)로 면제받지 않는 한 이 날에 미사에 참여할 의무가 있다. 고의로 이 의무를 이행하지 않는 사람은 대죄(peccatum grave)를 짓는다.

## 태그
`성사론`, `성체`, `교회법`, `sententia_certa`, `필수`, `절대`

## 주장 (Claims)
1. 주일과 의무 축일에 미사 참여는 의무이다
2. 고의로 미사 의무를 이행하지 않는 것은 대죄이다
3. 중대한 이유가 있으면 이 의무에서 면제될 수 있다
4. 미사는 온 교회 생활의 기초이다
5. 교회법은 미사 참여를 의무로 규정한다 (교회법 1247조)

## 부정 (Negates)
1. 미사 참여는 선택 사항일 뿐이다
2. 미사 불참은 죄가 되지 않는다
3. 개인 기도로 주일 미사를 대체할 수 있다
4. 교회의 예배 규정은 구속력이 없다
