# CCC-0402 — 원죄의 보편적 전달

| 항목 | 내용 |
|:---|:---|
| **ID** | CCC-0402 |
| **출처** | 가톨릭 교회 교리서 (Catechism of the Catholic Church) |
| **조항** | §402 |
| **교의 등급** | De Fide |
| **파문(Anathema)** | 아니오 (트렌트 공의회 원죄 교령에서 관련 파문 존재) |
| **연도** | 1992 |

## 원문 (요약)
> 모든 인간은 아담의 죄에 연루되어 있다. 성 바오로는 "한 사람의 불순종으로 많은 사람이 죄인이 되었다"(로마 5,19)고 확언한다. 원죄는 아담의 후손 모두에게 전달되며, 이로 인해 인류 전체가 죄의 상태에 놓이게 된다. 이는 모방이 아닌 전파(propagation)에 의한 것이다.

## 태그
`구원론`, `원죄`, `de_fide`, `절대`, `필수`

## 주장 (Claims)
1. 원죄는 아담으로부터 모든 인류에게 전달된다
2. 원죄의 전달은 모방이 아닌 전파(propagatio)에 의한다
3. 모든 인간은 태어날 때부터 원죄의 상태에 있다
4. 아담의 죄는 개인적 죄가 아닌 인류 전체의 상태를 변화시켰다
5. 원죄로 인해 인간 본성은 타락한 상태에 놓여 있다

## 부정 (Negates)
1. 인간은 본래 죄 없는 상태로 태어난다
2. 원죄는 아담 개인에게만 한정된다
3. 원죄는 단순한 모방(imitatio)에 의해 전해진다
4. 인간은 스스로 원죄 없이 존재할 수 있다
