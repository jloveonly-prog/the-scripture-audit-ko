# CCC-1996 — 은총은 무상의 선물

| 항목 | 내용 |
|:---|:---|
| **ID** | CCC-1996 |
| **출처** | 가톨릭 교회 교리서 (Catechism of the Catholic Church) |
| **조항** | §1996 |
| **교의 등급** | De Fide |
| **파문(Anathema)** | 아니오 (트렌트 칭의 교령에 관련 교의 정의 존재) |
| **연도** | 1992 |

## 원문 (요약)
> 우리의 칭의는 하느님의 은총에서 온다. 은총은 하느님이 우리에게 거저 주시는 호의이며, 그분의 생명에 참여하도록 우리를 부르고 도와주시는 무상의 선물이다. 은총은 초자연적이고 무상(gratuitus)이며, 인간의 자연적 능력과 공로로는 획득할 수 없는 것이다. 은총은 인간의 어떤 준비 행위나 업적에도 선행한다.

## 태그
`구원론`, `은총`, `칭의`, `de_fide`, `절대`

## 주장 (Claims)
1. 은총은 하느님이 거저 주시는 무상의 선물이다
2. 칭의의 근원은 하느님의 은총이다
3. 은총은 초자연적이며 인간의 자연적 능력으로 획득할 수 없다
4. 은총은 인간의 공로에 선행한다
5. 은총은 하느님의 생명에 참여하도록 부르는 것이다

## 부정 (Negates)
1. 인간은 자신의 노력과 공로만으로 은총을 획득할 수 있다 (펠라기우스주의)
2. 칭의는 인간의 행위에서 비롯된다
3. 은총은 인간이 마땅히 받을 보상이다
4. 인간의 준비 행위가 은총에 선행한다
