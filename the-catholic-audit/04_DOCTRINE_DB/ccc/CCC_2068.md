# CCC-2068 — 계명 준수로 구원

| 항목 | 내용 |
|:---|:---|
| **ID** | CCC-2068 |
| **출처** | 가톨릭 교회 교리서 (Catechism of the Catholic Church) |
| **조항** | §2068 |
| **교의 등급** | De Fide |
| **파문(Anathema)** | 예 (트렌트 공의회 칭의 교령 Canon 19-20) |
| **연도** | 1992 |

## 원문 (요약)
> 트렌트 공의회는 십계명이 의화된 사람들에게도 의무적이라고 가르친다. 또한 제2차 바티칸 공의회도 확인한다: "주교들은 … 신자들에게 그들의 구원을 위해 지켜야 할 교리의 각 부분, 곧 … 하느님의 영광과 인간의 영원한 행복을 위해 행해야 할 것과 피해야 할 것을 가르쳐야 한다."

## 태그
`구원론`, `계명`, `칭의`, `de_fide`, `필수`, `절대`

## 주장 (Claims)
1. 십계명은 의화된 사람들에게도 의무적이다
2. 계명 준수는 구원을 위해 필수적이다
3. 의화된 후에도 하느님의 계명을 지켜야 한다
4. 계명 위반은 구원을 위태롭게 한다
5. 행해야 할 것과 피해야 할 것 모두 구원에 관련된다
6. 십계명을 어기는 대죄를 짓고 회개하지 않으면 영원한 구원을 상실한다

## 부정 (Negates)
1. 의화된 사람은 계명 준수의 의무에서 면제된다
2. 믿음만으로 구원되며 행위는 무관하다 (Sola Fide의 절대적 해석)
3. 은총의 상태에 있으면 계명을 지키지 않아도 된다
4. 십계명은 구약의 율법이므로 신약 시대에는 적용되지 않는다
5. 한 번 구원받은 신자는 죄를 지어도 구원을 잃지 않는다

## 관련 충돌
- → CCC-1996 (은총은 무상의 선물 — 무상 은총과 계명 의무의 긴장)
- → CCC-0847 (교회 밖 구원 가능성: 계명을 모르는 자도 구원 가능)
- → CCC-2010 (공로: 행위를 통한 공로 체계)
