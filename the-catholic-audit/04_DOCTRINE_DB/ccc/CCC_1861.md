# CCC-1861 — 대죄와 은총 상실

| 항목 | 내용 |
|:---|:---|
| **ID** | CCC-1861 |
| **출처** | 가톨릭 교회 교리서 (Catechism of the Catholic Church) |
| **조항** | §1861 |
| **교의 등급** | De Fide |
| **파문(Anathema)** | 아니오 (트렌트 칭의 교령에서 관련 교의 정의 존재) |
| **연도** | 1992 |

## 원문 (요약)
> 대죄(peccatum mortale)는 성화 은총(gratia sanctificans)을 파괴하고, 사랑(caritas)을 빼앗으며, 회개하지 않으면 영원한 지옥의 벌을 가져온다. 대죄는 하느님과의 관계를 단절시키고, 은총의 상태에서 인간을 분리시킨다. 대죄가 성립하려면 중대한 내용, 충분한 인식, 완전한 동의가 요구된다.

## 태그
`구원론`, `은총`, `칭의`, `de_fide`, `절대`

## 주장 (Claims)
1. 대죄는 성화 은총을 파괴한다
2. 대죄는 사랑(caritas)을 빼앗는다
3. 회개하지 않은 대죄는 영원한 지옥의 벌을 가져온다
4. 대죄는 하느님과의 관계를 단절시킨다
5. 대죄의 성립 조건: 중대한 내용, 충분한 인식, 완전한 동의
6. 인간은 대죄를 통해 은총의 상태를 잃을 수 있다 (구원의 상실 가능성)

## 부정 (Negates)
1. 한번 받은 은총은 결코 잃을 수 없다 (일단 구원 영원 구원)
2. 죄로 인해 하느님과의 관계가 단절되지 않는다
3. 모든 죄는 동일한 무게를 갖는다 (대죄/소죄 구분 부정)
4. 성화 은총은 어떤 인간 행위로도 파괴될 수 없다

## 관련 충돌

- → CCC-1452 (완전한 통회: 고해성사 없이도 대죄 용서 가능)
- → CCC-2181 (주일 미사 의무: 미사 불참이 대죄)
