# CCC-1257 — 세례의 필수성

| 항목 | 내용 |
|:---|:---|
| **ID** | CCC-1257 |
| **출처** | 가톨릭 교회 교리서 (Catechism of the Catholic Church) |
| **조항** | §1257 |
| **교의 등급** | De Fide |
| **파문(Anathema)** | 아니오 (트렌트 세례 교령에 관련 파문 존재) |
| **연도** | 1992 |

## 원문 (요약)
> 주님 자신이 세례는 구원에 필수적이라고 확언하셨다(요한 3,5 참조). 그러므로 세례는 복음이 선포된 곳에서 복음을 들을 수 있었던 모든 사람에게 구원을 위해 필수적이다. 교회는 세례 외에 구원의 다른 수단을 알지 못한다. 그러나 하느님 자신은 성사에 매이지 않으신다.

## 태그
`성사론`, `세례`, `구원론`, `de_fide`, `필수`, `조건부`

## 주장 (Claims)
1. 세례는 구원에 필수적이다
2. 복음을 들을 수 있었던 모든 사람에게 세례는 의무이다
3. 교회는 세례 외에 구원의 다른 수단을 알지 못한다
4. 그러나 하느님은 성사에 매이지 않으신다 (예외 가능성 인정)
5. 세례의 필수성은 주님 자신의 말씀(요한 3,5)에 근거한다

## 부정 (Negates)
1. 세례 없이도 통상적으로 구원될 수 있다
2. 세례는 구원을 위한 선택 사항일 뿐이다
3. 내적 믿음만으로 세례 없이 충분하다 (통상적 경우)
4. 성사는 구원에 불필요한 외적 의식일 뿐이다
