# CCC-2357 — 동성애 (Homosexualitas)

| 항목 | 내용 |
|:---|:---|
| **ID** | CCC-2357 |
| **출처** | 가톨릭 교회 교리서 (Catechism of the Catholic Church) |
| **조항** | §2357 |
| **교의 등급** | Sententia Certa |
| **파문(Anathema)** | 아니오 |
| **연도** | 1992 |

## 원문 (요약)
> 동성애는 같은 성(性)의 남자들이나 여자들 사이의 관계를 말하며, 이들은 같은 성의 사람에게서 배타적이거나 주된 성적 끌림을 경험한다. 동성애 행위는 성경에서 중대한 타락으로 제시되었다. 전통은 항상 동성애 행위를 본질적으로 무질서한(intrinsece inordinati) 것으로 선언하였다. 이 행위는 자연법에 어긋나며, 생명의 선물에 대해 성행위를 닫아 놓고, 진정한 감정적·성적 보완성에서 비롯되지 않는다. 어떤 상황에서도 이를 승인할 수 없다.

## 태그
`윤리`, `성윤리`, `자연법`, `sententia_certa`, `금지`, `절대`

## 주장 (Claims)
1. 동성애 행위는 본질적으로 무질서하다 (intrinsece inordinati)
2. 동성애 행위는 자연법에 어긋난다
3. 동성애 행위는 어떤 상황에서도 승인될 수 없다
4. 동성애 행위는 생명의 선물에 대해 성행위를 닫아 놓는다
5. 성경은 동성애 행위를 중대한 타락으로 제시한다

## 부정 (Negates)
1. 동성애 행위는 도덕적으로 중립적이거나 허용된다
2. 동성애 행위는 특정 상황에서 승인될 수 있다
3. 자연법은 동성애 행위에 대해 아무런 판단도 내리지 않는다
4. 동성 간의 성적 결합도 혼인과 동등한 가치를 갖는다
5. 성경은 동성애에 대해 침묵한다
