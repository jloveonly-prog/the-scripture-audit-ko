# KS복음 구원관 분석 (행함이 있는 믿음과 조건부 구원)
KS님의 39개 영상을 분석한 구원론 결과 입니다.

이 문서는 특정 영상에서 주장하는 구원관(이하 'KS복음 구원관')을 요약하고 분석한 것입니다. 해당 구원관은 단순히 "믿습니다"라고 고백하는 것만으로 구원이 완성되고 영원히 보장된다는 이른바 '구원파적 구원관'이나 '한 번 구원은 영원한 구원'이라는 교리를 매우 강하게 비판하고 있습니다.

## 💡 영상에 나타난 구원관 핵심 요약표

| 구분 | 영상 속 구원론 (KS복음 구원관) | 상세 설명 및 근거 |
| :--- | :--- | :--- |
| **믿음과 행함** | 동전의 양면 (행함이 있는 믿음) | 입술의 고백(믿음)과 말씀에 대한 순종(행함)은 분리될 수 없음. 단, 가톨릭식의 '행위 구원(보속, 공로)'은 철저히 배격함. |
| **구원의 보장** | 구원을 잃어버릴 수 있음 | "한 번 구원은 영원한 구원이다"라는 주장은 사탄의 속임수이자 개소리라고 강하게 비판함. 타락하여 회개치 않으면 지옥에 감. |
| **자유의지** | 철저한 자유의지 강조 | 하나님은 인간에게 고귀한 '자유의지'를 주셨으며, 인간은 이 자유의지를 사용해 하나님께 100% 순종을 '선택'해야 함. '한 번 구원받으면 끝까지 보호받는다'는 주장을 전면 부정. |
| **구원의 완성** | 현재 진행형 (끝까지 견디는 자) | 믿는 순간 구원이 완성되는 것이 아니라, 날마다 회개하고 십자가를 지며, 마지막 환난의 때에 끝까지 인내해야 구원을 얻음. |
| **천국 가는 자** | 아버지의 뜻대로 행하는 자 | "주여, 주여" 한다고 가는 것이 아니라, 하나님의 말씀(뜻)대로 순종하고 거룩함의 열매를 맺는 소수의 사람만이 감. |
| **죄와 불법의 실체** | 세상 시스템에 대한 맹신과 순응 | 주류 언론, 정부, 거짓 과학(둥근 지구), 기성 교회의 교리를 맹신하는 '멍청함'이 큰 죄이며, 가장 치명적인 불법은 영과 육을 더럽히는 '백신(고무신) 접종'임. |
| **기성 교회 비판** | 거짓 목자의 가르침 배격 | 환난을 피할 수 있다는 '휴거' 교리나 '한 번 구원은 영원한 구원'을 외치며 진실(백신, 딥스테이트)에 침묵하는 주류 목사들을 따르면 지옥에 감. |

## 상세 분석

### 1. "주여 주여 하는 자마다 다 천국에 가는 것이 아니다"
영상에서는 마태복음 7장 21절을 수차례 인용하며, 입술로만 주님을 찾고 심지어 주의 이름으로 능력을 행해도 '하나님의 뜻대로 행하지 않고 불법(말씀을 어기는 죄)을 행하는 자'는 천국에 들어갈 수 없다고 경고합니다. 예수님을 믿는다면서 세상과 짝하여 육신의 정욕대로 살고 거룩함이 없다면, 그 믿음은 가짜(죽은 믿음)입니다.

### 2. 믿으면 구원의 완성인가? (행위 구원과의 차이)
영상은 "믿기만 하면 구원이다"라는 값싼 은혜의 복음을 매우 비판합니다. 진정한 믿음에는 반드시 **'말씀을 지켜 행하는 삶(순종)'**이 뒤따라야 합니다.
* **동전의 양면:** 입으로 예수님을 시인하는 것과 예수님의 말씀을 행하는 것은 손바닥과 손등처럼 뗄 수 없는 하나의 믿음입니다.
* **행위 구원 반대:** 이것이 가톨릭처럼 기도, 헌금, 고행 등으로 죄의 대가를 치러 영생을 얻으려는 '행위 구원'을 의미하는 것은 아닙니다. 인간의 공로가 아니라, 진짜 믿음이 있다면 자연스럽게 십자가를 지고 거룩함을 추구하는 행함이 나온다는 것입니다.
* **끝까지 견뎌야 함:** 구원은 한순간에 완성되는 것이 아니며, 죽을 각오(자기 부인, 자기 희생)를 하고, 지속적으로 죄를 회개하며, 끝까지 인내하여 믿음을 지키는 자만이 구원을 얻습니다.

### 3. 구원을 잃어버릴 수 있는가? ("한 번 구원은 영원한 구원" 비판)
구원을 잃어버릴 수 있다고 강력하게 주장합니다. 영상 제공자는 "한 번 구원은 영원한 구원"이라는 교리를 **"개소리", "사탄의 미혹", "구원파적 이단 교리"**라며 격멸합니다. 히브리서 6장 말씀을 인용하여, 한 번 성령에 참여하고 하늘의 은사를 맛본 자라도 타락하여 떨어져 나가면 다시 회개에 이르게 할 수 없으며 지옥에 간다고 경고합니다. 즉, 교회를 오래 다니고 직분을 받았더라도, 육체의 열매를 맺고 불법을 행하면 하나님의 나라를 유업으로 받을 수 없습니다.

### 4. 자유의지와 '한 번 구원은 영원한 구원' 부정
영상은 인간의 **'자유의지(Free Will)'**를 절대적으로 강조하며, **'한 번 구원받으면 끝까지 보호받는다'**는 주장을 전면 부정합니다.
* 하나님은 오직 사람에게만 자유의지를 주셨고, 인간은 이 자유의지를 사용하여 하나님의 뜻에 100% 순종할지 말지를 스스로 '선택'해야 합니다.
* 선악과 사건 역시 인간이 자유의지를 발휘한 결과이며, 마지막 환난의 때(짐승의 표, 백신 등)에도 본인의 자유의지로 세상의 안락함을 버리고 좁은 길을 택해야만 구원받을 수 있다고 설명합니다.

### 5. '행함'과 '불법'의 구체적 기준 = 백신 거부와 세상 시스템 비판

영상에서 규정하는 가장 치명적인 **'불법'**은 바로 **'코로나19 백신(영상 내 은어: 고무신) 접종'**과 **'세상 시스템(딥스테이트, 주류 언론, 정부)에 대한 맹신'**이다.

| 항목 | 영상 내 주장 |
| :--- | :--- |
| **백신의 영적 위치** | 짐승의 표에 준하는(짐승의 표 시즌 2) 영적 더럽힘으로 봄 |
| **순응의 결과** | 입으로 아무리 주여 주여 외쳐도 세상에 순응하여 백신을 계속 맞는 자들은 지옥에 간다고 경고 |
| **진정한 순종의 의미** | 목숨이나 생계를 걸고서라도 통제 시스템(마스크, 백신 등)을 거부하는 것을 포함 |

### 6. '죄'와 '회개'의 새로운 정의 = 멍청함 인정과 영적 각성

KS복음에서 말하는 가장 큰 **'죄'** 중 하나는 사탄이 지배하는 세상의 권위자들(정치인, 전문가, 목사 등)을 의심 없이 믿고 따르는 **'멍청함(무지)'**이다.

| 항목 | 영상 내 주장 |
| :--- | :--- |
| **진정한 회개란** | 자신이 세상에 세뇌당해 멍청했음을 하나님 앞에서 철저히 인정하는 것 |
| **영적 각성의 내용** | 지구가 평평하다는 진실이나 백신의 해악성 등을 깨닫는 것을 동반해야 함 |
| **백신 접종자의 회개** | 백신을 맞았더라도 이것이 악한 것임을 뒤늦게 깨닫고 하나님 앞에 철저히 회개하면 용서받을 수 있다고 가르침 |

### 7. 기성 교회와 목회자의 구원 부정

"오직 믿음"이나 "한 번 구원은 영원한 구원"을 가르치는 주류 기독교단과 목사들을 **사탄의 하수인이자 위선자**로 강하게 정죄한다.

| 항목 | 영상 내 주장 |
| :--- | :--- |
| **기성 목사들의 문제** | 진짜 범죄(소아성애, 딥스테이트의 악행, 백신 접종 등)에는 침묵하면서 술·담배 같은 가벼운 죄로 교인들을 통제하고 가스라이팅하는 위선자들 |
| **결과** | 이들을 따르면 다 함께 지옥에 간다고 경고 |
| **거짓 교리의 예시** | '환난을 피할 수 있다'는 휴거 교리, '한 번 구원은 영원한 구원' 교리 등 |

### 결론 요약
이 영상들의 구원론은 철저히 **자유의지에 기반한 조건부 구원(인내와 순종 필수)**에 가깝습니다. 입술의 고백만으로는 구원받을 수 없고, 끊임없는 회개와 말씀에 대한 순종, 거룩한 행실이 동반되어야 하며, 세상과 타협해 불법을 저지르면 언제든 지옥에 갈 수 있다는 무겁고 엄중한 구원관을 제시하고 있습니다.

신학적 뿀대(1~4번)에 더해, 이 구원관의 **실천적 특수성(5~7번)**은 '순종'과 '불법'을 **백신 거부, 딥스테이트 세계관, 기성 교회 배격**이라는 구체적 음모론적 프레임에 직결시킨다는 점에서, 일반적인 조건부 구원론(C그룹 웨슬리안 등)과도 결정적으로 구별된다.
